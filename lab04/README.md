### Базовый датчик случайных чисел

**Задание:**
- реализовать базовый датчик случайных чисел;
- вычислить выборочные среднее и дисперсию:
  - для реализованного датчика;
  - для встроенного генератора языка программирования;
- размер выборки — **100 000 значений**;
- сравнить результаты с теоретическими;
- сделать вывод.

### Теория
В коде реализован мультипликативный конгруэнтный генератор псевдослучайных чисел по следующим формулам:

$$
x_i^* = (\beta x_{i-1}) \bmod M,
$$

$$
x_i = \frac{x_i^*}{M}, i = 1,2,\dots
$$

с параметрами:
- $$x_0$$ - стартовое значение
- $$\beta$$ - множитель (константа генератора)
- $$M$$ - модуль
- $$x_i$$ - случайное число в интервале $$(0, 1)$$

Теоретические значения взяты из известных свойств равномерного распределения на интервале $$(0, 1)$$:

- Среднее = $$0.5$$  
- Дисперсия = $$\frac{1}{12}$$

### Реализация базового датчика
```python
class BaseGenerator:
    def __init__(self, b, m, seed):
        self.b = b          # множитель β
        self.m = m          # модуль M
        self.x = seed       # текущее значение

    def rand(self):
        # Формула: x_i* = (β * x_{i-1}*) mod M
        self.x = (self.b * self.x) % self.m
        # Нормализация: x_i = x_i* / M
        return self.x / self.m

    def generate(self, n):
        numbers = []
        for _ in range(n):
            numbers.append(self.rand())
        return numbers
```

### Параметры генератора
```python
b = 2**32 + 3              # β = 4 294 967 299
m = 2**63                  # M = 9 223 372 036 854 775 808
seed = b                   # x₀* = β
N = 100_000                # объём выборки
```

### Расчёт статистик
```python
def count(numbers):
    n = len(numbers)
    mean = sum(numbers) / n           # выборочное среднее
    var = 0
    for x in numbers:
        var += (x - mean) ** 2
    var = var / n                     # выборочная дисперсия
    return mean, var
```
