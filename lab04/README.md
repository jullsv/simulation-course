## Базовый датчик случайных чисел

**Задание:**
- реализовать базовый датчик случайных чисел;
- вычислить выборочные среднее и дисперсию:
  - для реализованного датчика;
  - для встроенного генератора языка программирования;
- размер выборки — **100 000 значений**;
- сравнить результаты с теоретическими;
- сделать вывод.

### Реализация

Согласно этому методу псевдослучайная последовательность вычисляется по рекуррентым формулам:

$$
x_i^* = (\beta x_{i-1}) \bmod M,
$$

$$
x_i = \frac{x_i^*}{M}, i = 1,2,\dots
$$

где:
- $$x_0$$ - стартовое значение (`seed`)
- $$\beta$$ - множитель (константа генератора)
- $$M$$ - модуль
- $$x_i$$ - случайное число в интервале $$(0, 1)$$


```python
def rand(self):
        #формула вычисления значения последовательности
        self.x = (self.a * self.x) % self.m
        #приведение к интервалу (0, 1)
        return self.x / self.m
```

В следующем блоке показано вычисление выборочного среднего и дисперсии:
```python
def calculate(numbers):
    n = len(numbers)
    mean = sum(numbers) / n
    var = 0
    for x in numbers:
        var += (x - mean) ** 2
    var = var / n
    return mean, var
```    
### Теоретические значения

Для равномерного распределения на интервале $$(0, 1)$$:  
- Среднее = $$0.5$$  
- Дисперсия = $$\frac{1}{12}$$
