### Метод конечных разностей для уравнения теплопроводности

**Задание:**  
Реализовать моделирование изменения температуры в пластине на основе одномерного уравнения теплопроводности с использованием метода конечных разностей.

Выполнить моделирование с различными шагами по времени и по пространству.  
Заполнить таблицу значений температуры в центральной точке пластины после 2 секунд модельного времени.

| Шаг по времени, с \ Шаг по пространству, м | 0.1 | 0.01 | 0.001 | 0.0001 |
|-------------------------------------------|-----|------|-------|--------|
| 0.1 |70 |19,6851 |19,8615 |19,8633 |
| 0.01 |70 |19,694 |19,8715 |19,8733 |
| 0.001 |70 |19,695 |19,8725 |19,8743 |
| 0.0001 |70 |19,695 |19,8726 |19,8744 |

Параметры модели:
- Материал: сталь (ρ = 7800 кг/м³, c = 460 Дж/(кг·°C), λ = 46 Вт/(м·°C))
- Граничные условия: T(0) = -100°C, T(L) = 70°C
- Начальное условие: T₀ = 20°C
- Время моделирования: t = 10 с

В ходе выполнения лабораторной работы был использован метод конечных разностей (метод сеток) для численного решения одномерного уравнения теплопроводности.

***Вывод***
При времени моделирования t = 10 с температура в центральной точке пластины (x = 0.05 м) составляет ≈ 19.87°C (начальная была 20°C). Изменение всего на 0.13°C объясняется умеренной теплопроводностью стали (λ = 46 Вт/(м·°C)) — за 10 секунд тепло от границ не успевает существенно повлиять на температуру в центре. При измельчении сетки наблюдается хорошая сходимость результата (разница между dx=0.001 и dx=0.0001 составляет всего 0.002°C).
Для практических расчетов достаточно шага 0.001 м, обеспечивающего точность до сотых долей градуса при приемлемом времени вычислений.
